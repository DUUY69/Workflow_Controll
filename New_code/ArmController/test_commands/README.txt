Thư mục này chứa các file test JSON mẫu. Cách dùng:

1) Chạy ArmController:
   - cd ..
   - python .\arm_controller.py

2) Khi cần test, copy các file JSON trong thư mục này vào thư mục inbox\
   - Có thể dùng file copy_all_to_inbox.bat để copy nhanh

Các file:
 - upload_move_lua.json        -> Upload lua_scripts\MoveToMotor.lua
 - run_move_lua.json           -> Chạy MoveToMotor.lua
 - upload_db_activate.json     -> Upload và kích hoạt TechPoint_db\web_point_2.db
 - upload_db_no_activate.json  -> Upload DB nhưng không kích hoạt
